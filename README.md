# 🛡️ Swiggy Clone - Backend (Spring Boot)

A robust REST API backend for the Swiggy Clone application, managing food items, restaurants, and user authentication.

## ✨ Features
- **RESTful APIs:** Complete CRUD operations for Food, Users, and Restaurants.
- **Database:** Integrated with MySQL for persistent data storage.
- **Architecture:** Decoupled architecture to support multiple clients.
- **Data Security:** Structured DAO and Service layers.

## 🛠️ Tech Stack
- **Framework:** Spring Boot (Java)
- **ORM:** Spring Data JPA / Hibernate
- **Database:** MySQL
- **Build Tool:** Maven

## 🚀 Getting Started
1. Clone the repo: `git clone <https://github.com/Badhusha01/Swiggy-Clone-Backend>`
2. Update `application.properties` with your MySQL credentials.
3. Run as Spring Boot App in Eclipse/IntelliJ.
